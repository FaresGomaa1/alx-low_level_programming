# 0x00-hello_world

Welcome to the `0x00-hello_world` directory of the `alx-low_level_programming` repository!

## Overview

This directory is dedicated to projects and exercises related to the basics of the C programming language. The focus is on understanding how the compiler works, from preprocessing to compilation, linking, and execution.

## Files in this Directory

- `0-preprocessor`: A script that preprocesses a C file and saves the result into another file.
- `main.c`: A simple "Hello, World!" C program showcasing basic structure.
